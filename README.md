# فوروارد پیامک (SMS Forwarder)

یک اپلیکیشن اندرویدی که پیامک‌های دریافتی از شماره‌های مشخص را به‌طور خودکار به یک شماره‌ی دیگر فوروارد می‌کند.

## امکانات
- تعریف چند رول: «از شماره X → به شماره Y» (یا فوروارد از **همه‌ی** شماره‌ها)
- ذخیره‌ی دائمی رول‌ها روی گوشی (حتی بعد از بستن یا حذف برنامه از رم)
- یک سوییچ کلی برای فعال/غیرفعال کردن کل سیستم فوروارد
- فعال/غیرفعال کردن هرکدام از رول‌ها به‌صورت جداگانه، یا همه‌شان با یک دکمه
- اجرا در پس‌زمینه با یک Foreground Service + نوتیفیکیشن ثابت، به‌همراه ری‌استارت خودکار بعد از روشن شدن گوشی، تا برنامه تا حد امکان توسط سیستم بسته نشود

## ⚠️ محدودیت مهم درباره‌ی «همیشه باز بودن»
اندروید به‌خاطر مدیریت باتری، در برخی گوشی‌ها (مخصوصاً شیائومی، هوآوی، سامسونگ، اوپو و...) اپ‌هایی که در پس‌زمینه هستند را می‌بندد، حتی با Foreground Service. برنامه از کاربر می‌خواهد که «بهینه‌سازی باتری» را برایش غیرفعال کند، ولی برای اطمینان کامل باید در تنظیمات گوشی (بخش‌هایی مثل «Battery > App launch» یا «Autostart») هم این اپ را دستی مجاز/Whitelist کنید. این محدودیتِ خودِ اندروید است و هیچ اپی نمی‌تواند ۱۰۰٪ آن را دور بزند.

## چرا APK آماده ندادم؟
برای ساخت APK به Android SDK و Gradle و دسترسی اینترنت به سرورهای گوگل نیاز است که در محیط اجرای من در دسترس نیست. برای همین کل پروژه را به‌صورت سورس‌کد کامل و آماده‌ی بیلد آماده کردم. دو راه ساده برای گرفتن APK داری:

### روش ۱ (پیشنهادی، بدون نصب هیچ‌چیزی روی سیستم خودت): GitHub Actions
1. یک ریپازیتوری جدید در GitHub بساز و همه‌ی این پروژه را در آن push کن.
2. به‌محض push شدن، تب **Actions** ریپازیتوری را باز کن؛ یک workflow به اسم «Build APK» خودش اجرا می‌شود.
3. بعد از پایان (چند دقیقه طول می‌کشد)، روی همان اجرا کلیک کن و از بخش **Artifacts** فایل `sms-forwarder-apk` را دانلود کن؛ داخلش `app-debug.apk` است.
4. آن را روی گوشی‌ات نصب کن (باید نصب از منابع ناشناس/Unknown sources را برای مرورگر یا فایل‌منیجرت فعال کنی).

### روش ۲: Android Studio (لوکال)
1. آخرین نسخه‌ی [Android Studio](https://developer.android.com/studio) را نصب کن.
2. پوشه‌ی `SmsForwarder` را با «Open» در Android Studio باز کن.
3. صبر کن Gradle Sync تمام شود (اگر پیغام «Gradle wrapper missing» آمد، گزینه‌ی استفاده از Gradle نصب‌شده یا ساخت خودکار wrapper را تایید کن).
4. گوشی‌ات را با یواس‌بی وصل کن (Developer Options + USB debugging فعال باشد) یا از یک شبیه‌ساز استفاده کن، و روی دکمه‌ی ▶ Run بزن.

## نحوه‌ی استفاده در برنامه
1. برنامه را باز کن؛ مجوزهای SMS و نوتیفیکیشن را تایید کن.
2. با دکمه‌ی + یک رول جدید بساز: شماره‌ی فرستنده (یا تیک «همه‌ی شماره‌ها») و شماره‌ی مقصد را وارد کن و ذخیره کن.
3. با سوییچ بالای صفحه کل سیستم فوروارد را روشن/خاموش کن، و با سوییچ کنار هر رول، همان رول را جدا فعال/غیرفعال کن. دکمه‌های «فعال کردن همه» و «غیرفعال کردن همه» هم برای کار دسته‌جمعی هستند.
4. پیامک روی گوشی مقصد به این شکل می‌رسد: «پیامک از [شماره فرستنده]: [متن پیام]»

## نکته
فوروارد پیامک از طریق SIM خودِ گوشی انجام می‌شود، پس هزینه‌ی ارسال پیامک طبق تعرفه‌ی اپراتور خودت محاسبه می‌شود. همچنین این ابزار را فقط برای گوشی و شماره‌ی خودت یا با رضایت صاحب شماره استفاده کن.
